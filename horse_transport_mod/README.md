# Horse Transport Mod for Minecraft 1.20.1 Forge

## Описание
Этот мод добавляет в Minecraft сущность "Коневоз" (Horse Transport), которая позволяет:
- Загружать любых животных внутрь коневоза
- Животные "приклеиваются" к полу и остаются на месте во время движения
- Игрок может сесть за руль и управлять коневозом как машиной
- Выгрузить животное в любом месте

## Установка
1. Установите Minecraft Forge для версии 1.20.1
2. Скопируйте файл мода `.jar` в папку `mods` вашего Minecraft
3. Запустите игру с профилем Forge

## Использование
1. Создайте коневоз с помощью команды: `/summon horse_transport:horse_transport ~ ~ ~`
2. Чтобы загрузить животное:
   - Подведите животное близко к коневозу (в радиусе 5 блоков)
   - ПКМ по коневозу - животное будет загружено внутрь
3. Чтобы сесть за руль:
   - ПКМ по пустому коневозу (или с уже загруженным животным)
4. Управление:
   - W - движение вперед
   - S - движение назад/торможение
   - A/D - поворот влево/вправо
5. Чтобы выгрузить животное:
   - Слезьте с коневоза
   - ПКМ по коневозу с зажатой клавишей Shift (или просто ПКМ если внутри есть животное)

## Структура проекта
```
horse_transport_mod/
├── build.gradle          # Конфигурация сборки Gradle
├── gradle.properties     # Настройки Gradle
├── settings.gradle       # Настройки проекта
└── src/main/
    ├── java/com/example/horse_transport/
    │   ├── HorseTransportMod.java      # Основной класс мода
    │   ├── HorseTransportEntity.java   # Сущность коневоза
    │   └── ModEvents.java              # Обработчики событий
    └── resources/
        ├── mods.toml                   # Метаданные мода
        └── horse_transport.mixins.json # Конфигурация mixins
```

## Сборка
Для сборки мода выполните:
```bash
./gradlew build
```
Собранный файл появится в `build/libs/`
